# 🛒 ქართული ფასების შემდარებელი

ყველა ქართული სუპერმარკეტისა და აფთიაქის ფასების შედარება — AI-ს დახმარებით.

---

## ⚡ გაშვება (ნაბიჯ-ნაბიჯ)

### ნაბიჯი 1 — Claude API გასაღების მიღება

1. გახსენი [console.anthropic.com](https://console.anthropic.com)
2. შედი ან დარეგისტრირდი
3. "API Keys" → "Create Key"
4. გასაღები დააკოპირე (გამოიყურება ასე: `sk-ant-api03-...`)

---

### ნაბიჯი 2 — .env ფაილის შექმნა

1. გახსენი `backend` საქაღალდე
2. `.env.example` ფაილი დაარქვი `.env`-ად  
   *(მარჯვენა კლიკი → Rename)*
3. გახსენი `.env` ნოუთბუქით და ჩაწერე:
   ```
   ANTHROPIC_API_KEY=sk-ant-api03-შენი-გასაღები-აქ
   ```

---

### ნაბიჯი 3 — სერვერის გაშვება

`price-finder` საქაღალდეში ორჯერ დააწკაპუნე **`start.bat`** ფაილზე.

ან Command Prompt-ში:
```
cd C:\Users\user\price-finder
start.bat
```

---

### ნაბიჯი 4 — ფრონტენდის გახსნა

`frontend\index.html` ფაილი გახსენი **Chrome** ან **Edge** ბრაუზერით.

## ❓ პრობლემა?

**"ANTHROPIC_API_KEY not found"** — `.env` ფაილი ვერ იძებნება, ან გასაღები არ ჩაწერია.

**"Connection refused"** — სერვერი არ მუშაობს. გაუშვი `start.bat`.

**"No results"** — მაღაზიის საიტი შეიცვალა. AI ავტომატურად გამოტოვებს ამ მაღაზიას.
